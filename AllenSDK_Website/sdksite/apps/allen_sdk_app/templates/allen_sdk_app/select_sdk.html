The Brain Observatory SDK contains data collected from mouse visual cortex. This data features visually evoked neuronal activity from: fluorescing neurons in selected cortical layers neurons in different visual areas neurons in different mouse Cre lines The data will allow you to observe the dynamic behavior of neurons

The Cell Types SDK database contains data from comprehensive study on individual neurons in mouse primary visual cortex. This data features the following neuronal characteristics: electrophiological properties Morphological attributes The data will allow you to discover the characteristic differences between types of neurons

The Mouse connectivity database contains a large assortment of experimental data collected in order to better understand how specific regions in the mouse brain are connected. The data consists of a collection of high resolution images, each of which targets a specific area of the mouse brain and follows the pathways the respective region projects onto.
