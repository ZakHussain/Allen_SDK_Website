The Brain Observatory SDK contains data collected from mouse visual cortex. This data features visually evoked neuronal activity from:
  • Fluorescent neurons in selected cortical layers.
  • Neurons in different visual areas.
  • Neurons in different mouse Cre lines.
Here, you can observe the dynamic behavior of neurons!
The Cell Types SDK database contains data from comprehensive study on individual neurons in mouse primary visual cortex. This data features the following neuronal characteristics:
  • Electrophysiological properties.
  • Morphological attributes.
Here, you will discover the characteristic differences between types of neurons!
The Mouse connectivity Atlas contains experimental data to better understand how connections in the mouse brain. The data consists of the following:
  • a collection of high resolution images.
  • images targeting specific area mouse cortex.
  • images following the neuoronal pathways and projections.
Here, you can explore the pathways through which neurons connect!
