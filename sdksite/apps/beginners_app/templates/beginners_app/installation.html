Installing

Anaconda Download

Downloading the Anaconda distribution is one of the simplest ways to install python plus a number of libraries popular in data science

follow this link to download the Anaconda, Python 2.7 distribution for your operating system

PIP Installation

Having downloaded the Anaconda distribution, you now have access to 'pip'

run the following in the command line:

pip install allensdk

if you needed to uninstall the allensdk, run the following line

pip uninstall allensdk

The Required Dependencies that you must have are

  • NumPy
  • SciPy
  • matplotlib
  • h5py
  • pandas
  • pynrrd
  • jinja2

Operating Systems Supported

  • Mac
  • Windows
