About the Brain Observatory

The Brain Observatory is home to data that comes from visual regions of mouse cortex. There are two way of accessing the data. You can either search by Experiment or by Cell.

So how do you decide which method of search to use?

Use the Experiment search if you would like to find neuronal-population responses to stimuli.

Use the Cell search if you would like to find single-neuron responses to stimuli.

An in depth explanation of how to do each of these searches is found under the Tutorials section. For now, let's get you oriented with our experimentation techniques so you understand what our data means

Overview of the Experiments

The data we gathered comes from numerous experiments. Each experiment follows similar experimental design-- a transgenic mouse is stimulated with three synthetic stimuli and two natural stimuli. The key difference between the collection of experiments is that the following variables are changes:

  • the transgenic mouse line
  • the area of the visual cortex
  • depth at which the cells are images

Why manipulate these variables?

  • Having transgenic mice allow us to observe fluorescence in cells of different cortical layers.
  • There are six areas of the visual cortex, so we want to see the unique response of neurons in each region
  • observing the activtiy of cells at different depths creates a more distinct picture of how visual info is processed
